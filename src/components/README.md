## 🧠 Key Concepts

### 1. Component

* A React component is a JavaScript function
* It returns JSX

Example:

```jsx
function TaskInput() {
  return <div>UI</div>;
}
```

---

### 2. JSX

* JSX looks like HTML but is actually JavaScript
* It describes UI structure

---

### 3. Rendering Flow

Step-by-step:

1. React calls component function
2. Function returns JSX
3. JSX is converted to Virtual DOM
4. React updates real DOM

---

## 🔄 Code Flow

```text
App.jsx
  ↓
Render TaskInput
  ↓
TaskInput() runs
  ↓
Return JSX
  ↓
React renders UI
```
